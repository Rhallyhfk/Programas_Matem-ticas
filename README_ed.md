# EDITOR DE GRAFOS
Programa para visualizacion de grafos (Matematicas discretas, Esctructuras de datos).

Abrir la carpeta princial, y ejecutar con este comando

compilación: 

- javac -d bin -cp "lib/flatlaf-3.6.jar" src/*.java

correr el programa:

- java -cp "bin;lib/flatlaf-3.6.jar" GrafoTest 


El progrma es util para la generacion de caminos con dijsktra, Árboles de expancion y recorridos de grafos (BFS, DFS).
